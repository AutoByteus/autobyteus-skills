# Long-form depth standard

Use this when drafting the final `article.md`.

Goal: make `deep-research-article` default to a genuinely substantial long-form artifact, not a compressed explainer that merely points at sources.

## Default length policy

- If the user explicitly specifies a target length, follow it.
- If the user does **not** specify a target length:
  - normal deep-research article: target roughly **5,000-8,000 words**
  - complex / contested / policy / theology / strategy topics: target roughly **7,000-12,000 words**
  - only go materially shorter if the user explicitly asks for a brief, memo, or summary

This does not mean “inflate the word count.”
It means the article should have enough room to:
- define terms,
- surface competing interpretations,
- synthesize evidence in prose,
- show reasoning step by step,
- and spell out implications and limits.

## Body expectations

- The body should carry the argument, not the references section.
- Do not write “see sources below” as a substitute for explanation.
- Each roadmap move normally needs multiple substantive paragraphs, not a single paragraph plus bullets.
- Integrate quotes, data, and evidence synthesis directly into the body with citations embedded in prose.
- Keep a references section at the end, but treat it as documentation of the argument, not the place where the argument lives.

## Paper-grade default

When the user says `deep research`, assume:
- more detail than a blog post,
- more developed evidence integration than a sermon outline,
- and more explicit handling of objections, uncertainty, and scope limits.

Unless the user asks for a short version, the final artifact should read closer to a serious long-form paper or whitepaper than to a short article summary.

## Anti-patterns

Fail the draft if it looks like any of these:
- an outline lightly expanded into prose
- a 3-6 move structure where each move gets only one short paragraph
- most evidence deferred to the references section instead of integrated into the body
- a short “executive summary + takeaways” piece dressed up as deep research
- thin reasoning followed by a long source list
