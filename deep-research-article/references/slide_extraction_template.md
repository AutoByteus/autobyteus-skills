# slide_extraction.md template (article → slide units)

Make one row per slide. Keep the slide headline as a **claim**.

| Slide | Headline (claim) | Evidence anchor (exact quote/data) | Supporting bullets (2–4) | Visual suggestion (location + hero symbol + props) | Text budget |
|---:|---|---|---|---|---|
| 1 |  |  |  |  | short |
| 2 |  |  |  |  | medium |
| 3 |  |  |  |  | heavy |

Text budget meanings:
- `short`: title + 2–3 bullets
- `medium`: title + 1–2 quote lines + 3–5 bullets
- `heavy`: needs a dense layout (two-column text panel) or should be split
