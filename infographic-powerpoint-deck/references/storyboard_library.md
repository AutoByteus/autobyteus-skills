# Storyboard library (make the deck feel like a narrative)

Use this to give the whole deck a “beginning → tension → climax → resolution” flow, even in teaching decks.

## Default progression (12-slide deck)

1) **Establishing** (wide shot): set place + theme (harbor/city silhouette, big symbol).
2) **Map/structure** (top-down): orient the audience (timeline/cards).
3) **Covenant love** (soft dusk): bride + cross + shield.
4) **Discernment** (candle desk): three-card framework.
5) **Contrast** (stage spotlight): scale packaging vs truth.
6) **Integrity** (workshop): scissors cut chain; opportunity removed.
7) **Warning** (night stained glass): mask + halo + shadow.
8) **Confrontation** (street tension): control strings + broken chain.
9) **Reframe** (heritage→scars): comparison cards.
10) **Climax** (montage): storm/road/bars—endurance list.
11) **Heart** (night map): concern + escape scene.
12) **Resolution** (dawn): shield + checklist; calm confident light.

## Prompt add-on lines (paste per slide)

- “本页在整套叙事中的位置：第 [N] 幕（[Establishing/Warning/Climax/Resolution]）。画面必须体现对应情绪强度。”
- “镜头：采用 [wide/medium/close-up/top-down/diagonal tension]；保持与前后页的连续性。”

## Intensity ladder (keep viewers awake)

- Slides 1–2: 低→中（引入、定位）
- Slides 3–6: 中（建立对比与张力）
- Slides 7–10: 中→高（警告与高潮）
- Slides 11–12: 高→收束（情感与应用）
