# Stage 6 Code Review

## Purpose

Run the independent code-review gate after Stage 5 evidence is current.

## Enter This Stage When

- Stage 5 is complete for the current iteration
- `workflow-state.md` has moved to Stage 6 and locked source edits

## Stage-Owned Outputs

- `tickets/in-progress/<ticket-name>/code-review.md`

## Exit Gate

Leave Stage 6 only when the review decision is `Pass` and the mandatory review checks are satisfied.
